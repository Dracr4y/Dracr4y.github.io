在github上部署hexo博客

  1. 需要安装的工具：Git，Node.js
  2. 右键打开git
  3. 全局安装hexo
  4. 在想要安装的文件夹下打开git，执行hexo init
  5. 安装依赖
  6. 启动服务
  7. 访问localhost:4000,显示以下内容即成功
  8. 注册github账号
  9. 创建仓库【名字后边应跟上.github.io】
  10. 配置ssh key，连续三次回车，会生成密钥在本地
  11. 打开复制的内容
  12. 再打开你的github设置，把刚才复制的内容放进去，标题随便填
  13. 测试是否成功
  14. 配置用户名和邮箱
  15. 配置hexo的_config.yml文件【记得加空格！！！】
  16. 安装一个插件（不然会报错）
  17. 现在可以上传你的网站到github了，执行hexo d
  18. 访问你的github page网址
  19. Done
